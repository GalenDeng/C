## 链表的基本操作 (2018.1.9)
## `空链表`
```
List list = new ArrayList();
此时的list就是一个空链表，此时它里面什么都没有，但是已经开辟了一块空间
“为什么还有构造空链表”
这就是要为了以后向里面放元素放的准备，
打个比方，你想要存放东西，那么首先你得有仓库吧，而你自己没有，你就得先申请一个空的仓库
已备你以后向里放你想放的东西～～
```
