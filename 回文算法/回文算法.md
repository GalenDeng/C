## 回文算法 -- vc6.0平台实验 (2018.1.9)
1. `回文串`
``` 所谓回文串，简单来说就是正着读和反着读都是一样的字符串，比如abba，noon等等，一个字符串的最长回文子串即为这个字符串的子串中，是回文串的最长的那个
```
2. `题目`
```
求所有不超过200的n值，n的平方是具有对称性质的回文数。所谓回文数就是将一个数从左向右读与从右向左读是一样的。例如：3443和1234321都是回文数 
```
